FAQ по использованию Server Scanner:

  • Для сканирования сети, Вам следует ввести диапазон адресов в основное окно и нажать кнопку "Сканировать".
    Если Вам требуется приостановить поиск, нажмите кнопку "Приостановить"
  • Результат сканирования отобразится ниже поискового окошка.
  • При нажатии на кнопку "Файл" отображается диалоговое окно, с выбором открытия или сохранения результатов сканирования.
  • При нажатии на кнопку "Сканирование" отображается чекбокс с выбором вариантов сканирования: FTP, HTTP или общее сканирование, при выборе обеих опций.
